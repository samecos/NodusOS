# Nodus 交互设计稿 (Wireframes & Mockups)

> 版本: v0.1 | 日期: 2026-05-04

---

## 1. 设计总纲

- **画布**：全屏（无桌面图标、无任务栏、无窗口管理器）
- **核心元素**：代码工作区（中心）、呼吸灯（右下边缘）、意图输入条（按需浮现）、结果卡片（按需浮现）
- **色彩**：深色主题为主，卡片半透明毛玻璃效果
- **字体**：代码区等宽字体（JetBrains Mono / SF Mono），UI区系统字体

## 3. 语音唤醒流程

### 3.1 唤醒词检测

用户说 "Nodus"：

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                    ┌─────────────────────┐                   │
│                    │        ...          │                   │
│                    └─────────────────────┘                   │
│                                                              │
│                                                              │
│              ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐                   │
│               Nodus 正在听...                                 │
│              │         ▓▓▓▓▓▓▓▓▓▓        │                   │
│                        ▓▓▓▓▓▓▓▓▓▓                             │
│              └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘                   │
│                    (语音波形条)                                │
│                                                              │
│                                                          ◉   │  ← 呼吸灯 (Listening: 脉动动画)
└──────────────────────────────────────────────────────────────┘
```

**交互**：
1. 用户说"Nodus" → 唤醒词检测
2. 呼吸灯变亮 + 开始脉动动画（#00ff88, 80% opacity, 1.5s周期）
3. 屏幕中央下方浮现半透明语音波形条
4. 用户继续说意图："refundOrder被哪些地方调用了"
5. 波形条随声音变化
6. 声音结束（VAD检测）→ 波形条收拢 → 转写中...

## 4. 文字输入流程（Ctrl+Space）

### 4.1 输入条浮现

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                    ┌─────────────────────┐                   │
│                    │        ...          │                   │
│                    └─────────────────────┘                   │
│                                                              │
│                                                              │
│              ┌─────────────────────────────────┐            │
│              │ 🔤 想问什么？                     │            │
│              └─────────────────────────────────┘            │
│                                                              │
│                                                          ○   │
└──────────────────────────────────────────────────────────────┘
```

**交互**：
1. 用户按 Ctrl+Space → 屏幕中央偏下浮现输入条
2. 输入条自动聚焦，placeholder "想问什么？"
3. 用户输入文字意图，回车发送
4. 输入条收起 → 呼吸灯变Working → 结果卡片浮现

### 4.2 无声模式切换

长按Ctrl+Space 2秒（或说出/Nodus 进入无声模式）：

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│              ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐                   │
│               🔇 已进入无声模式                               │
│              │  语音唤醒已停止              │                   │
│               Ctrl+Space 随时唤起输入                          │
│              └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘                   │
│                                                              │
│                                                          ◌   │  ← 呼吸灯 (无声模式: 半透明)
└──────────────────────────────────────────────────────────────┘
```

提示条3秒后自动消散。呼吸灯变为半透明（#888888, 10% opacity）。
