## 2. 语义索引流程

### 2.1 全量索引

```
         IndexStarted event
                │
                ▼
         ┌─────────────┐
         │ 遍历文件树    │
         │ (glob匹配)   │
         └──────┬──────┘
                │
                ▼
         ┌─────────────┐
         │ 过滤: 支持的  │
         │ 扩展名+排除  │
         │ node_modules │
         │ 等目录       │
         └──────┬──────┘
                │
                ▼
    ┌───────────────────────┐
    │  对每个文件循环:       │
    │                       │
    │  1. 读取文件内容       │
    │  2. 计算 checksum     │
    │  3. 比较上次索引的     │
    │     checksum          │
    │     ├─ 相同 → SKIP    │
    │     └─ 不同 → CONTINUE│
    │  4. Tree-sitter 解析  │
    │     ├─ 成功 → 继续    │
    │     └─ 失败 → 记录    │
    │        ParseError,    │
    │        SKIP           │
    │  5. 提取符号 (Symbol) │
    │  6. 提取引用 (Ref)    │
    │  7. Upsert到DB       │
    │  8. 更新file_index   │
    │     _state表          │
    │  9. 发出 IndexProgress│
    │                       │
    └───────────┬───────────┘
                │
                ▼
    ┌───────────────────────┐
    │  全文件解析完成后:      │
    │  1. 构建调用图         │
    │     CallGraphBuilder  │
    │  2. 存储到 Knowledge  │
    │     Store             │
    │  3. 发出 IndexReady   │
    └───────────────────────┘
```

### 2.2 增量索引（文件变更时）

```
    FileWatcher detects change
    (文件创建/修改/删除)
                │
                ▼
         ┌─────────────┐
         │ 防抖 500ms   │
         │ (合并连续变更) │
         └──────┬──────┘
                │
                ▼
         ┌─────────────┐
         │ 检查变更类型  │
         └──────┬──────┘
                │
    ┌───────────┼───────────┐
    ▼           ▼           ▼
┌───────┐  ┌───────┐  ┌───────────┐
│Created│  │Modified│  │Deleted    │
└───┬───┘  └───┬───┘  └─────┬─────┘
    │          │              │
    ▼          ▼              ▼
┌───────┐  ┌───────┐  ┌───────────┐
│=新增  │  │=修改  │  │ 从DB中     │
│文件   │  │文件   │  │ 删除该文件  │
│索引   │  │先删除 │  │ 的所有符号  │
│       │  │旧符号 │  │ 和引用     │
└───┬───┘  │再索引 │  └─────┬─────┘
    │      └───┬───┘        │
    │          │            │
    └──────────┼────────────┘
               │
               ▼
        ┌─────────────┐
        │ 重建受影响   │
        │ 的调用图边   │
        │ (该文件涉及  │
        │  的所有调用  │
        │  关系)       │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │ 发出         │
        │ IndexFileDone│
        │ IndexStatus  │
        │ → Ready      │
        └─────────────┘
```
